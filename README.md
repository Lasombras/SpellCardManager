Spell Card Manager
==================

Spell Card Manager for pathfinder RPG

-- RCP Project use Eclipse JUNO 4.2.2
     1. Install RCP Package in "Install New Software..."
     2. Download RCP Delta Package here : http://download.eclipse.org/eclipse/downloads/drops4/R-4.2.2-201302041200/
     3. Unzip the archive file. It should contain an eclipse directory with one feature and several plugins.
     4. Close Eclipse.
     5. Overlay your Eclipse installation with the files from the archive. Replace any files when prompted.
     6. Restart Eclipse. You may need to use the -clean option.
     7. Under the Preferences menu item, select Plug-in Development | Target Platform. Click the Reload button.
	(At step 7, uncheck the “build target platform based on the target’s installed plug-ins” option before reloading. Otherwise the delta pack features will not be shown.)
     8. Ensure that the list of plug-ins includes plugins for platforms other than your operating system. (i.e. Ensure that the list includes org.eclipse.equinox.launcher.carbon.macosx, org.eclipse.equinox.launcher.gtk.linux.x86, org.eclipse.equinox.launcher.win32.win32.x86, etc).If these plug-ins aren’t listed, try clicking the Reset button.


-- Android Project use Android 7
	1. Install Android Eclipse Plugins
	2. More info on Google Android Site here : http://developer.android.com/sdk/installing/installing-adt.html
